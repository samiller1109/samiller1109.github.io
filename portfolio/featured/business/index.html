The World’s Oldest Businesses

A SQL & Python Data Storytelling Project

Project Summary

This project explores the world’s oldest continuously operating businesses using structured data analysis and visualization. The dataset, obtained from BusinessFinancing.co.uk, includes information on centuries-old companies spanning industries and continents. The goal was to uncover global trends in business longevity using SQL and Python.

Technologies Used

  • Python: For data preprocessing, SQL integration, and visualization using Pandas, Seaborn, and Matplotlib.
  • SQLite: An in-memory SQL database was used to run efficient relational queries across three normalized tables: businesses, categories, and countries.
  • Google Colab: Enabled interactive development and sharing of the analysis notebook.

Key Analyses

  • Calculated the range of founding years to identify the earliest and latest companies in the dataset.
  • Identified the oldest business and its industry using SQL joins.
  • Isolated companies founded before 1000 AD and visualized them along a color-coded timeline.
  • Ranked the most common industries for centuries-old companies globally and per continent.
  • Highlighted the oldest company on each continent through an inverted scatter plot showing founding years.

Visualizations

A range of charts were created to tell the data’s story:

  • Timeline of Founding Years: A horizontal line plot showing the span between the oldest and most recent businesses in the dataset.
  • Bar Charts: Top industries by company count, both globally and per continent.
  • Scatter Plot: Oldest businesses per continent, inverted to emphasize historical age.
  • Color-Coded Timeline: Businesses founded before 1000 AD, individually marked for clarity and visual appeal.

Conclusion

This project demonstrates how structured query language (SQL) and Python-based data science tools can uncover compelling historical insights. From identifying ancient Japanese inns to mapping industry resilience over centuries, this analysis reveals how tradition, geography, and adaptability have helped some companies endure for over a thousand years.
