The World’s Oldest Businesses

A SQL & Python Data Storytelling Project

Project Summary

This project explores the longevity of historic businesses around the globe using data from BusinessFinancing.co.uk. It analyzes industry distributions, continental trends, and founding year timelines with SQL and Python visualizations.

Key Visualizations

Timeline showing the earliest and latest founding years.

Bar chart of the most common industries among the oldest companies.

Scatterplot of the oldest companies per continent.

Sample SQL Query

SELECT b.business, b.year_founded, c.category
FROM businesses b
JOIN categories c ON b.category_code = c.category_code
ORDER BY b.year_founded ASC
LIMIT 1;

What I Learned

  • Designing and querying a normalized in-memory SQLite database.
  • Joining and aggregating multi-table CSV data efficiently.
  • Communicating data insights with Seaborn and Matplotlib.
